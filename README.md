PrepBuddy is a RAG-based (Retrieval-Augmented Generation) AI study companion designed to help Class 12 students understand their subjects more effectively.
It currently supports Biology and is built to scale easily to other subjects like Physics, Chemistry, etc., using a simple .env configuration.

Students can ask questions in natural language, and PrepBuddy retrieves the most relevant study content and generates accurate, contextual answers using Groq LLMs, delivered through an interactive Streamlit interface.
